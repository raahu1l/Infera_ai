# My-Infera Expo App

## About The Project

My-Infera is a mobile app built with Expo and React Native that connects to a backend hosted on Replit for AI-powered features. This app provides a smooth UI and seamless communication with the backend API.

## Features

- Cross-platform support (Android, iOS, Web)
- Powered by React Native and Expo SDK 54
- Connects to backend hosted on Replit for AI tasks
- Clean and modular codebase
- Easy setup and development

## Technologies Used

- Expo SDK 54
- React Native 0.81
- React 19.1
- React Native Web
- Node.js (backend on Replit)

## Getting Started

### Prerequisites

- Node.js installed
- Expo CLI or use `npx` to run Expo commands
- Expo Go app or Expo Dev Client app installed on device
